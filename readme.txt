                    CS2 STARS MANAGER v1.0
════════════════════════════════════════════════════════════════

ТРЕБОВАНИЯ:
  • Node.js 14 или выше (скачать с https://nodejs.org/)
  • Интернет для первого запуска (установка зависимостей)

УСТАНОВКА:
  1. Установите Node.js если ещё не установлен
  2. Распакуйте все файлы в любую папку
  3. Двойной клик на start.bat
  4. При первом запуске подождите ~30 сек (установка зависимостей)

ИСПОЛЬЗОВАНИЕ:
  • Двойной клик на start.bat для запуска
  • Следуйте инструкциям в консоли
  • Файлы сессий хранятся в папке sessions/
