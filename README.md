# CS2 Armory Shop Manager

CLI‑утилита на Node.js для управления покупками в CS2 Armory и контроля баланса звёзд на нескольких Steam‑аккаунтах.
![Screenshot](screen.jpg)

- Получение награды Армори не заходя в игру
- Работа с несколькими аккаунтами
- Скорость получения награды в разы выше чем руками
